# Laravel E-Commerce

## 📌 نظرة عامة
مشروع **Laravel E‑Commerce** هو تطبيق متجر إلكتروني متكامل تم بناؤه باستخدام **Laravel** و **MySQL** وBlade Templates، مع خصائص مثل:
- نظام تسجيل دخول وتسجيل مستخدمين
- عرض المنتجات والإضافة إلى السلة
- تعديل وإرسال الطلبات
- لوحة تحكم للأدمن لإدارة المنتجات والطلبات
- حماية وأمان عالي  
تم تصميم هذا المشروع ليكون قاعدة قوية لأي متجر إلكتروني قابل للتطوير بسهولة.

## 📦 المتطلبات  
قبل تشغيل المشروع تحتاج إلى:
- PHP >= 8.x  
- Composer  
- MySQL  
- Node.js + NPM (لـ Laravel Mix/Front‑End)
